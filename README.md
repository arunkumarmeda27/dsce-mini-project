A centralized, web-based platform designed to digitize the lifecycle of academic mini-projects at **Dayananda Sagar College of Engineering (DSCE)**. 

---

## 🚀 Overview
The **DSCE Mini Project Management System** replaces manual spreadsheets and emails with a structured workflow. It connects Students, Faculty Guides, and Administrators to ensure transparency and efficient project tracking.

## ✨ Key Features

### 👨‍🎓 Student Portal
* **Group Management:** Form project groups and invite team members.
* **Proposal Submission:** Register project titles and domains.
* **Document Vault:** Upload reports, PPTs, and source code.
* **Feedback Loop:** Real-time notifications on guide approvals.

### 👨‍🏫 Guide Portal
* **Supervision Control:** Accept or reject project requests based on domain expertise.
* **Evaluation:** Review submissions and provide actionable feedback.
* **Progress Tracking:** Monitor the development status of all assigned groups.

### 🛡️ Administrator Portal
* **User Management:** Approve/Reject student and guide registrations.
* **Allocation Logic:** Manually or automatically assign guides to project groups.
* **Data Export:** Generate reports for departmental documentation.

---

## 🛠️ Technology Stack
* **Frontend:** React.js, Tailwind CSS (for responsive UI)
* **Backend:** FastAPI (Python-based high-performance API)
* **Database:** Firebase Firestore (NoSQL)
* **Auth:** Firebase Authentication
* **Version Control:** Git & GitHub

---

## 🔄 System Workflow
1. **Registration:** Users sign up; Admin verifies institutional credentials.
2. **Proposal:** Students form groups and submit project abstracts.
3. **Allocation:** Admin assigns a Faculty Guide to the group.
4. **Validation:** Guide reviews the project scope and accepts the group.
5. **Submission:** Students upload milestones (Reports/PPTs).
6. **Final Review:** Guide provides final approval and marks.
